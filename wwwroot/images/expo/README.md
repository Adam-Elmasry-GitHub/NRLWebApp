# Expo Demo QR Code

## Required File

Place your Google Forms feedback QR code as:
- **File name:** `feedback-qr.png`
- **Location:** `/wwwroot/images/expo/`
- **Recommended size:** 300x300 pixels

## Purpose

This QR code will be displayed on the completion screen after users finish all expo demo tasks.
It should link to a Google Form or feedback survey where users can provide their experience feedback.

## Creating the QR Code

1. Create a Google Form for expo feedback
2. Get the shareable link
3. Generate a QR code from the link using a QR code generator
4. Save as `feedback-qr.png` in this directory
